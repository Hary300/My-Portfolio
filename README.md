# 🌐 Developer Portfolio

A modern, responsive developer portfolio built with Next.js, TypeScript, and Tailwind CSS. Designed with a clean user interface, smooth animations, and reusable component architecture to showcase professional experience, technical skills, featured projects, and contact information. The project emphasizes performance, accessibility, responsive design, and maintainable code using modern React development practices.

## ⚙️ Getting Started

### Clone Repository

```bash
git clone https://github.com/Hary300/My-Portfolio.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Start Production Server

```bash
npm run start
```
